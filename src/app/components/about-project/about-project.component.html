About the application:
Technology used:
  • HTML 5
  • CSS3
  • BootStrap 5
  • Angular Js 16
Methods used to do operations:
  • Save Method
  • Edit or Update Method
  • Delete Method
  • View the entered log Method
Save Method

Here, We can Save user's [Name, Phone Number, Email Id] in to the Register log. and store it in Directory interface type.

For user Id reference I used getRandomId Method to generate random number for the user Id.

View Users Log Method

Below the Register UI part, you can view the saved user's contact details like Name, Phone Number, Email Id-- and also you can see the edit and delete button in each entry.

Here you can edit or delete the entry by clicking on the buttons provided.

Delete Method

You can delete the entered details by clicking on the delete button provided in each entry.

Here, I done the delete by getting Id of the particular entry you are clicking, and I matching with the directory.

And, removing that part alone from the directory.

Edit or Upadte Method

You can edit or update your entry by clicking on edit button provided in each entry.

Here, I match the edit Id with the entry's Id you are clicked , if it get matched the entry will reflect in register UI and the save button will turn into update button.

So, you can do the necessary updation and click update button to push the updated entry into the directory.
